Installation
===============================

#### npm
```shell
$ npm i
$ gulp watch
```

#### bower
```shell
$ bower install
$ gulp watch
```

Open up Chrome Devtools, go to chrome://extensions and add 'manifest.json' in the file explor (filepath should be app/manifest.json)
